android 去掉dialog 的title

(2012-03-05 18:03:09)
转载▼
标签：

dialog

android

title

杂谈

 分类： android
android的Dev Guide中说Dialog不能去掉title部分.实际上是可以去除的.方法如下:

Dialog d = new Dialog(context);
d.requestWindowFeature(Window.FEATURE_NO_TITLE);
